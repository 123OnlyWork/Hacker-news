# Hacker News 

## Запуск проекта

### Клонировать репазиторий

```
git clone https://github.com/123OnlyWork/Hacker-news.git

```
### Установка зависимостей ###
Перейдите в директорию front :
```
cd front

```
### Установить зависимости с помощью Yarn ###
```
yarn install

```
### Запуск ###

```
yarn start

```
